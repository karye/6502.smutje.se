Steg 5 — LCD-display

Mål

Koppla in en LCD1602 med I²C-backpack som visar aktuell adress och instruktion i realtid — direkt på kopplingsdäcket, utan datorskärm.

Nya komponenter

AntalKomponent
1LCD1602 med I²C-backpack (adress 0x27 eller 0x3F)
4Kopplingstrådar (VCC, GND, SDA, SCL)

Kopplingar

LCDArduino
GNDGND
VCC5V
SDAPin 20 (SDA)
SCLPin 21 (SCL)

Kopplingsschema

Arduino-kod

#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);  // eller 0x3F

void setup() {
  // ... befintlig setup från steg 1-4 ...
  lcd.init();
  lcd.backlight();
  lcd.setCursor(0, 0);
  lcd.print("W65C02S Ready");
}

void pulse() {
  // ... befintlig klockpuls-kod ...

  // Uppdatera LCD efter varje klockcykel
  lcd.setCursor(0, 0);
  lcd.print("A:$"); lcd.print(addr, HEX);
  lcd.print("    ");
  lcd.setCursor(0, 1);
  lcd.print("D:$"); lcd.print(data, HEX);
}

Testning

  1. LCD:n ska visa "W65C02S Ready" vid uppstart
  2. Vid varje knapptryck (steg 4) uppdateras displayen med aktuell adress och data
  3. Kör en I²C-skanner om displayen inte svarar — testa både 0x27 och 0x3F
✅ Förväntat: LCD:n visar adress och instruktion i realtid. Fullt fristående testbänk!
