Steg 4 — Manuell stegning med knappar

Mål

Lägg till två tryckknappar för manuell stegning. Knapp 1 matar en klockcykel per tryck. Knapp 2 kör fram till nästa instruktion (SYNC).

Nya komponenter

AntalKomponent
2Tryckknappar
1Kopplingstråd (SYNC → D7)

Kopplingsschema

Arduino-kod

#define SYNC 7
#define BTN_CLK   5
#define BTN_INSTR 6

// ... tidigare setup från steg 3 ...

void setup() {
  // ... befintlig kod ...
  pinMode(SYNC, INPUT);
  pinMode(BTN_CLK,   INPUT_PULLUP);
  pinMode(BTN_INSTR, INPUT_PULLUP);
}

void loop() {
  // Knapp 1: Enskild klockcykel
  if (!digitalRead(BTN_CLK)) {
    delay(50);
    while (!digitalRead(BTN_CLK));
    delay(50);
    pulse();
  }

  // Knapp 2: Kör tills nästa instruktion (SYNC hög)
  if (!digitalRead(BTN_INSTR)) {
    delay(50);
    while (!digitalRead(BTN_INSTR));
    delay(50);
    do { pulse(); } while (!digitalRead(SYNC));
  }
}

Testning

  1. Knapp 1: En klockcykel per tryck
  2. Knapp 2: Kör tills SYNC går hög (ny instruktion)
✅ Förväntat: Programmet kan stegas manuellt, en klockcykel eller en instruktion i taget.
