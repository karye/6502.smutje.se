Steg 3 — Databuss och minnesemulering

Mål

Koppla in databussen och RWB. Arduinon agerar minnesemulator — levererar instruktioner från en array. Vi laddar ett NOP-testprogram på $8000.

Nya komponenter

AntalKomponent
8100 Ω motstånd (databuss-skydd)
8Lysdioder (databuss-visualisering)
8220 Ω motstånd (LED-strömbegränsning)

Kopplingsschema

⚠ Busskollision! När RWB = LÅG driver CPU:n bussen. Sätt PORTA till INPUT innan PHI2 går hög.

Arduino-kod

#define PHI2  2
#define RESB  4
#define RWB   3
uint8_t mem[65536];

void setup() {
  Serial.begin(115200);
  pinMode(PHI2, OUTPUT); pinMode(RESB, OUTPUT); pinMode(RWB, INPUT);
  DDRK = 0x00; DDRF = 0x00;

  mem[0xFFFC] = 0x00; mem[0xFFFD] = 0x80;  // reset → $8000
  mem[0x8000] = 0xEA;  // NOP

  digitalWrite(RESB, LOW);
  for (int i = 0; i < 5; i++) pulse();
  digitalWrite(RESB, HIGH);
  for (int i = 0; i < 10; i++) pulse();
}

void pulse() {
  digitalWrite(PHI2, LOW); delay(10);
  uint16_t a = (PINK<<0)|(PINF<<8); bool rw = digitalRead(RWB);
  if (rw) { DDRA=0xFF; PORTA=mem[a]; }
  else    { DDRA=0x00; }
  digitalWrite(PHI2, HIGH); delay(10);
  if (!rw) mem[a] = PINA;
  Serial.print(rw?"R":"W"); Serial.print(" $"); Serial.println(a, HEX);
}

void loop() { pulse(); }

Testning

  1. Serieövervakaren visar $FFFC → $FFFD → $8000 (NOP-loop)
  2. Databuss-LED:arna visar 11101010 (NOP = $EA)
✅ Förväntat: CPU:n läser reset-vektor, hoppar till $8000, exekverar NOP i loop. Full minnesemulering!
