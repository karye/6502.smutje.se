Steg 1 — Strömmatning och grundläggande klocka

Steg 1 Steg 2 Steg 3 Steg 4 ← Översikt

Mål

Verifiera att W65C02S-processorn får spänning och att klockpulser genereras korrekt. Vi driver klockan från Arduino D2 och tänder en lysdiod i takt med klockpulsen.

Viktigt! Placera 100 nF-kondensatorn så nära CPU:ns pinnar 8 och 21 som möjligt. Utan den kan processorn bete sig oberäkneligt även vid låga hastigheter.

Arduino-kod

// Steg 1 — Blinka PHI2 på D2 med 1 Hz
#define PHI2 2

void setup() {
  pinMode(PHI2, OUTPUT);
}

void loop() {
  digitalWrite(PHI2, HIGH);
  delay(500);   // 500 ms hög → 1 Hz
  digitalWrite(PHI2, LOW);
  delay(500);   // 500 ms låg
}

Testning

  1. Ladda upp koden till Arduinon
  2. Kontrollera att lysdioden blinkar med jämn takt (1 Hz = en blinkning per sekund)
  3. Mät med multimeter: 5 V mellan VDD (pin 8) och VSS (pin 21)
  4. Kontrollera att alla 5 pull-up-pinnar har ~5 V mot VSS

Förväntat resultat

✅ Lysdiod blinkar stadigt 1 gång per sekund. Detta bekräftar att CPU:n får ström och att klocklinjen fungerar. Processorn gör inget meningsfullt ännu (adress- och databuss är inte inkopplade), men den är spänningssatt och klockad — redo för steg 2.
